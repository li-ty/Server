### Windows命令
| 命令 | 用法 |
| :-----: | :----: |
| netstat -ano | 查看全部端口占用 |
| netstat -ano \| findstr "端口号" | 查特定端口 |
| taskkill /f /t /im "进程id或者进程名称" | 杀死进程 |
| sc create fastgithub binPath= "C:\Program Files\fastgithub\fastgithub.exe" | 将exe注册成windows服务 |
### Linux命令
| 命令 | 用法 |
| :-----: | :----: |
| lsof -i:端口号 | 查特定端口 |
| kill -9 端口号 | 杀死进程 |
